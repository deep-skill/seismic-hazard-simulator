Summary of crustal area source model inputs:

coords:
- Coordinates exported from GIS for the geometry of each crustal source zone

logic_trees:
- Logic tree images for each different group of crustal area source zones

Excel files:
UAS_AI_C_FullCat_20231204.xlsx:
- Input parameters for the source zones: Crust above interface (Central Shallow) and Crust above interface (Central Deep)
- Branching matches the corresponding logic tree in the logic_trees folder

UAS_CC_WC_C_FullCat_20231204.xlsx:
- Input parameters for the source zones: Coastal Cordillera (Central) and Western Cordillera (Central), i.e. sources containing the project sites
- Branching matches the corresponding logic tree in the logic_trees folder

UAS_OtherZones_FullCat_20231204.xlsx:
- Input parameters for all other source zones
- Branching matches the corresponding logic tree in the logic_trees folder
